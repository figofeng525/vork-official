# VORK Official Website

Official website prototype for VORK, a standardized functional fitness race series for city athletes, gyms, teams, and partners.

## Local development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

The site is built with Next, React, Tailwind CSS, Vinext, and the Sites-compatible Cloudflare Worker output.
